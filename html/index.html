YelpCamp

YelpCamp is a Node.js application using mainly Express, EJS, Mongoose, and Passport. Its intention is to post about different campsites, leaving likes, comments, etc.

TinyBlog

TinyBlog is a Pharo/Smalltalk application using Seaside (Web), Voyage (Mongo), Bootstrap (Twitter), and Magritte (Seaside Components) frameworks. It contains blog entries that can be filtered and an administrative page for adding, editing, and deleting posts.

SimpleReddit

SimpleReddit is a Pharo/Smalltalk application using Seaside (Web), Glorp (Object-Relational Mapping), and PostgreSQL. Its purpose is for users to post links that get voted up or down.
